Плагин выпадающего календаря для выбора даты

Календарь полностью кастомизируется по оформлению, имеет несколько готовых тем. Кроме этого имеется ряд настроек для более гибкого использования, а также возможность локализации. Изначально имеется русская и английская локализация, но можно добавлять любые другие по не сложному шаблону.

Есть несколько режимов работы календаря для упрощения выбора любой произвольной даты:

  • режим выбора даты (с прокруткой по месяцам)
  • режим выбора месяца (с прокруткой по годам)
  • режим выбора года (с прокруткой по 10 лет)

Переключение режимов осущетсвляется кликом по заголовочной части календаря. Прокрутка - кликом по навигационным кнопкам (влево и вправо).

Если в поле для ввода даты уже введена дата в формате "DD.MM.YYYY", то при вызове календаря он откроется на месяце введенной даты.

Примеры:

1) Календарь с настройками по умолчяанию.

2) Календарь с кастомизированными настройками и темой оформления "theme-zn-blue".

Подключение файлов скриптов и стилей

Сначала подключаем JQuery, затем скрипт плагина и стили:

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1/jquery.min.js"></script>
<script type="text/javascript" src="js/jquery.mssDatePicker.js"></script>
<link rel="stylesheet" href="css/mssDatePicker.css">

Если будут использоваться готовые темы оформления, то подключаем CSS файл с темами:

<link rel="stylesheet" href="css/mssDatePicker-themes.css">

Установка HTML-кода на страницу

<div class="mss-datepicker">
  <input>
  <button></button>
</div>

Если нужно разместить несколько календарей с разными настройками, то дополнительно присваиваем каждому блоку свой id.

Если нужно использоваться готовую тему оформления, то добавляем контейнеру класс темы оформления. 

<div class="mss-datepicker theme-zn-blue">

Готовые темы оформления:

Тема Пример
theme-zn-blue
theme-table

Подключение Javascript

$(document).ready(function () {
  $('.mss-datepicker button').on('click', function(){
    mssDatePicker({
     lang: 'ru'
    }, this);
  });
});

Если нужно разместить несколько календарей с разными настройками, то вместо селектора
".mss-datepicker" прописываем нужный селектор id и задаем настройки для каждого календаря отдельно.

Настройки

Свойство Тип значения Значение по умолчанию Описание
lang string "en" Язык локализации (изначально имеется русская - "ru" и английская - "en" локализация)
weekStart number 0 С какого дня начинается неделя (0 - Вс, 1 - Пн и т.д.)
dateSeparator string "." Разделитель групп разрядов даты при выводе в поле
yearInHead boolean true Отображать в заголовке календаря год
markWeekends boolean false Выделять выходные дни красным цветом (если true, то выходным дням в календаре присваивается класс ".mss-weekend")
markToday boolean true Выделять сегодняшний день (если true, то сегодняшнему дню в календаре присваивается класс ".mss-today")
monthShort boolean true Использовать в режиме выбора месяца укороченные названия месяцев (3-4 символа)
holidays array[string] - Массив праздничных дней в формате:
  • номер дня недели (0 - Вс)
  • либо дата в формате "D.M" - сработает каждый год
  • либо дата в формате "D.M.YYYY" - сработает 1 раз
  • либо диапазон дат в формате "D.M.YYYY-D.M.YYYY"
Выбранным дням в календаре присваивается класс ".mss-holiday")
desabled array[string] - Массив отключенных дней (которые нельзя выбрать) в формате:
  • номер дня недели (0 - Вс)
  • либо дата в формате "D.M" - сработает каждый год
  • либо дата в формате "D.M.YYYY" - сработает 1 раз
  • либо диапазон дат в формате "D.M.YYYY-D.M.YYYY"
Выбранным дням в календаре присваивается класс ".mss-desabled")
